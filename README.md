# BartlebysCore

**BartlebysCore** is the core Engine of [**BartlebyKit**](https://github.com/Bartlebys/BartlebyKit). It is written is Swift 4.x.

## Supported platforms :

- macOS
- iOS
- Linux
- tvOS (partial)
- watchOS (partial)
